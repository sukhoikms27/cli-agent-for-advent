# 06 — VectorMath (`rag/VectorMath.kt`)

Реализовано вручную (без внешних либ — FAISS/NumPy в JVM-CLI избыточен для малого корпуса).

## cosine(a, b): Float
`a·b / (‖a‖·‖b‖)`. Защита: пустой/нулевой вектор → 0 (нет деления на 0); разная длина → сравнение
по общей префиксной длине. Возвращает [−1, 1].

## topK(query, chunks, k): List<ScoredChunk>
Линейный скан: чанки с непустым `embedding` → `(chunk, cosine(query, it))` → sort по убыванию →
take(k). Чанки без вектора skip. Меньше K кандидатов → возвращает сколько есть. `k=0` → пусто.

Производительность: десятки-сотни чанков — миллисекунды. Для 10K+ чанков день 22/будущее может
добавить ANN-индекс, но JSON+linear-scan достаточен для CLI-агента (AGENTS.md: «для CLI-агента
1-5 чатов, 10-100 сообщений JSON быстрее SQLite»).
